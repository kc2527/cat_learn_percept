# cat_learn_percept

Standalone PsychoJS experiment for perceptual assessment in
II category-learning studies.

## Start here

1. Start local server (`make serve-open`).
2. Inspect geometry in `space_viewer.html`.
3. Work to dial in PA and CP parameters and trial counts for a single participant/session/day.
4. Details of PA and CP parameter definitions are here:
    - `/Users/mq20185996/Dropbox/cat_learn_percept/docs/CP_protocol.md`
    - `/Users/mq20185996/Dropbox/cat_learn_percept/docs/PA_protocol.md`

## Local serving

Use the provided `Makefile`:

```bash
make serve
```

Stop server with `Ctrl+C`.

## Launcher

Use `launcher.html` to start sessions without manually editing URL params.
